# Cosmic Autofill

Open source autofill extension for Shopify


## Installation


```bash
Download and unzip
Go to chrome://extensions/
Enable Developer mode
Drag and drop the unzipped folder into the extensions page
```
    
## Features

- Fast Shopify Autofill
- Lightweight
- Cross platform
